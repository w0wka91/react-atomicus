# Calendar

The start date is adjustable

<Playground>
  <Calendar initDate={new Date(1990, 10)} />
</Playground>

The minimum/maximum date that a user can select can be restricted

<Playground>
  {() => {
    const [selectedDates, setSelectedDates] = useState([])
    return (
      <>
          <Calendar
            initDate={new Date(2019, 10)}
            minDate={new Date(2019, 10, 2)}
            maxDate={new Date(2019, 10, 10)}
            selectedDates={selectedDates}
            onSelect={date => {
              if(selectedDates.find(d => Dayjs(d).isSame(Dayjs(date), 'day'))) {
                setSelectedDates(selectedDates.filter(d => !Dayjs(d).isSame(Dayjs(date), 'day')))
              } else {
                setSelectedDates([...selectedDates, date])}
              }
            }
          />
      </>

    )

}}

</Playground>
