# Button

## Size

A button can have different sizes

<Playground>
  <ButtonWrapper>
    <Button size="small">Small </Button>
    <Button size="medium">Medium</Button>
    <Button size="large">Large</Button>
  </ButtonWrapper>

## Intent

A button can have different types of intent

<Playground>
  <ButtonWrapper>
    <Button intent="primary">Primary</Button>
    <Button intent="success">Success</Button>
    <Button intent="warning">Warning</Button>
    <Button intent="danger">Danger</Button>
  </ButtonWrapper>
</Playground>

## Hierarchy

A button can have a differnt place in the hierarchy

<Playground>
  <ButtonWrapper>
    <Button hierarchy="primary">Primary</Button>
    <Button intent="warning" hierarchy="secondary">
      Secondary
    </Button>
    <Button hierarchy="tertiary">Tertiary</Button>
  </ButtonWrapper>
</Playground>

## Fluid

A fluid button takes the width of its container

<Playground>
  <Button fluid>Fluid Button</Button>
</Playground>

## Icon

A button can contain an Icon

<Playground>
  <ButtonWrapper>
    <Button>
      <Button.Icon name="corner-down-left" />
    </Button>
    <Button intent="primary">
      <span>Continue</span>
      <Button.Icon name="arrow-right" />
    </Button>
    <Button hierarchy="secondary" intent="danger">
      <Button.Icon name="x" />
      <span>Delete</span>
    </Button>
  </ButtonWrapper>
</Playground>

### Loading

A button can show a loading indicator

<Playground>
  {() => {
    const [toggleValue, toggle] = useToggle(false)
    const loadingFunc = () => {
      toggle()
      setTimeout(toggle, 2000)
    }
    return (
      <>
        <Button loading={toggleValue} onClick={loadingFunc}>Click to load ...</Button>
      </>

)
}}

</Playground>
