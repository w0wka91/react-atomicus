# Icon

The icons that are provided by this library are coming from [feather-icons](https://feathericons.com/). Thus, all of the listed icons are available.

## Basic usage

<Playground>
  <Icon name="database" />
</Playground>

## Color

A icon can have different colors

<Playground>
  <Icon name="dollar-sign" color="green" />
</Playground>

## Size

The size of an icon is adjustable

<Playground>
  <Icon name="dollar-sign" color="green" size="4rem" />
</Playground>

## Fill

Fill the icon with a nice color

<Playground>
  <Icon name="flag" fill="red" size="4rem" />
</Playground>
